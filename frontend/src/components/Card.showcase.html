Card Component Showcase

A flexible card component with variants, elevation levels, and padding options

Card Variants

Cards support three variants: default, interactive, and bordered.

Default

Default Card

A simple card with subtle border and shadow. Use for static content displays.

Interactive (hover me)

Interactive Card

Enhanced hover effects with scale and shadow. Use for clickable cards or navigation.

Bordered

Bordered Card

Emphasized border for visual separation. Use when you need stronger visual distinction.

Elevation Levels

Cards support five elevation levels using shadow tokens: none, sm, md, lg, and xl.

None

No Shadow

Flat appearance without shadow elevation.

Small (sm)

Small Shadow

Subtle elevation for minimal depth.

Medium (md) - Default

Medium Shadow

Standard elevation for most cards.

Large (lg)

Large Shadow

Prominent elevation for emphasis.

Extra Large (xl)

Extra Large Shadow

Maximum elevation for dramatic effect.

Padding Options

Cards support three padding sizes: compact, default, and spacious.

Compact

Compact Padding

Minimal padding for dense layouts or small cards.

Default

Default Padding

Standard padding for balanced spacing and readability.

Spacious

Spacious Padding

Extra padding for emphasis or larger content areas.

Usage Examples

Basic Card

<Card>
  <h3>Card Title</h3>
  <p>Card content goes here</p>
</Card>

Interactive Card

<Card variant="interactive" onClick={handleClick}>
  <h3>Clickable Card</h3>
  <p>Click me to perform an action</p>
</Card>

Card as Link

<Card variant="interactive" href="/details">
  <h3>Navigation Card</h3>
  <p>Navigate to details page</p>
</Card>

Custom Padding and Elevation

<Card 
  variant="bordered" 
  padding="compact" 
  elevation="lg"
>
  <h3>Customized Card</h3>
  <p>With compact padding and large elevation</p>
</Card>
