Usage Guidelines:
  • xs/sm: Buttons, tags, keyboard shortcuts
  • md: Cards, dropdowns, standard panels
  • lg/xl: Modals, popovers, overlays
  • 2xl: Full-screen modals, critical overlays
  • header/sidebar: Navigation components for consistent elevation
  • inner: Pressed states, input fields

Usage Guidelines:
  • sm (0.25rem / 4px): Small elements like tags, badges
  • md (0.375rem / 6px): Default for buttons and inputs
  • lg (0.5rem / 8px): Cards and panels
  • xl (0.75rem / 12px): Large cards and modals
  • 2xl (1rem / 16px): Hero sections and prominent panels
  • full (9999px): Circular elements (avatars, badges)
