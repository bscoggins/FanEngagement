Transitions & Timing

Duration Tokens

--duration-fast
150ms

Quick interactions (hover, focus)

--duration-normal
200ms

Standard transitions (buttons, cards)

--duration-slow
300ms

Complex transitions (drawers, modals)

Easing Functions

--ease-in
cubic-bezier(0.4, 0, 1, 1)

Accelerating (entering animations)

--ease-out
cubic-bezier(0, 0, 0.2, 1)

Decelerating (exiting animations) ⭐ Most common

--ease-in-out
cubic-bezier(0.4, 0, 0.2, 1)

Smooth both ways (complex animations)

/* Standard transition pattern */ .button { transition: background-color var(--duration-normal) var(--ease-out), box-shadow var(--duration-normal) var(--ease-out); } /* Card hover with lift effect */ .card { transition: transform var(--duration-fast) var(--ease-out), box-shadow var(--duration-fast) var(--ease-out); } .card:hover { transform: translateY(-2px); }
